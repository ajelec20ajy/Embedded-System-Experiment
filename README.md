임베디드시스템실험 프로젝트입니다.

All_File는 전체 파일
Only_Main은 main.c만 포함합니다.

공유하는 코드가 많아 Zone1과 Zone2의 주석을 메인 주석으로 합니다.




프로젝트 주제 : 상태 기반 중앙 집중형 차량 제어기 + 에러 대응 처리

간단설명 : 4개의 중간 제어기가 각각에 할당받은 말단 제어기/센서 - 중앙 제어기의 통신을 관리하며, 중앙 제어기가 제공받은 데이터를 분석해 시스템 전체 상태를 결정하는 시스템. 기본적으로 페달을 통한 모터 드라이버 조작, 충돌 방지 시스템 등이 탑재되어있으며 15가지의 에러에 대응을 구현.

사용 보드 : LaspberryPi Zero 2W 1개 + STM32F103C8T6개발보드 * 9개

통신 : CAN(HAL, socketCAN), SPI(HAL), I2C(HAL)
